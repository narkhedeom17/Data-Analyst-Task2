## Project Overview
This project analyzes global sales data to identify key trends, revenue drivers, profit margins, shipping performance, and business opportunities.

## Pages Included
### 1. Sales Overview Dashboard
Contains KPIs:
- Total Sales
- Total Profit
- Total Orders
- Avg Unit Price
- Avg Shipping Days
Plus regional and category-wise visuals.

### 2. Detailed Analysis Dashboard
Shows trends, comparisons, revenue vs cost, profit %, and shipping analysis.

### 3. Insights & Recommendations
Business insights and actionable recommendations for improving performance.
